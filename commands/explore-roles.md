A well-defined developer role includes:
- **Responsibilities** - Day-to-day tasks and goals
- **Required Skills** - Core technical competencies

## Role Selection Guide

### Quick Assessment

**Question 1: Work Style Preference**
- [ ] I enjoy building user-facing features ‚Üí Frontend
- [ ] I prefer working on servers/APIs ‚Üí Backend
- [ ] I like infrastructure and automation ‚Üí DevOps
- [ ] I'm interested in data processing ‚Üí Data Engineer
- [ ] I want to lead teams ‚Üí Engineering Manager

**Question 2: Preferred Technology Level**
- [ ] Close to users/UI ‚Üí Frontend, UX Design
- [ ] Middle layer/logic ‚Üí Backend, Full-stack
- [ ] Infrastructure/systems ‚Üí DevOps, System Design
- [ ] Data/AI ‚Üí Data Engineer, ML Engineer

**Question 3: Career Goals**
- [ ] Individual contributor ‚Üí Specialist roles
- [ ] Technical leadership ‚Üí Architect, Staff Engineer
- [ ] People management ‚Üí Engineering Manager, Director
- [ ] Entrepreneurship ‚Üí Full-stack, Product

### Compatibility Matrix

| Goal | Role 1 | Role 2 | Role 3 |
|------|--------|--------|--------|
| Deep technical expertise | Backend | Architect | ML Engineer |
| Work-life balance | DevRel | Technical Writer | Product Manager |
| Fast job market entry | QA | Technical Writer | Frontend |
| Highest salary | ML Engineer | Principal Architect | DevOps |
| Remote opportunities | All modern roles | (Most) | (Most) |
