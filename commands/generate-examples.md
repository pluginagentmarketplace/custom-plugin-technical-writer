### Step 1: Run the Command
```
/generate-examples
```

Provide:

- **Common Task** - Typical operation
- **Advanced Usage** - Complex scenario
- **Integration** - API/service integration

- C#/.NET
- PHP

- Or multiple languages

- Complete working code

- Related links
